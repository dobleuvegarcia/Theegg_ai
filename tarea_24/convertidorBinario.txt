En este ejercicio valido el numero en una interfaz comprobando k solo se puedan
introducir numeros enteros o decimales index.py.
hay que instalar la libreria pillow:

windows:
$ python -m pip install pillow

si tienes anaconda la instruccion es:
conda install -c anaconda pillow
si aun asi no consigues instalar la libreria pillow preguntame y estare encantada de ayudarte a instalarla.

para poder ver la imagen de fondo

mas tarde en la funcion redondeo el numero a un entero para obtener el bianrio del entero.
y muestro el array de restos del reves para obtener el binario de ese numero redondeado.
